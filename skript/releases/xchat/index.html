xChat

Created by: _iZ_

Released Jan. 24, 2022

xChat is a fully-Skripted chat management script that has 6 different chat moderation options to help manage your server.

Important

The options section of xChat contains an option for the name of the Skript file. Make sure to change that option if you change the name of the file.

BE SURE TO READ THE README.TXT FILE!!!

Dependencies (1)

• Skript - https://github.com/SkriptLang/Skript/releases

Features

• Mute/unmute chat

• Clear chat

• Chat filter

• Chat link filter

• Caps limit

• Message cooldown

• Permissions for all commands and sub-commands

• Bypass permissions

• Customization for all options

• Customization for most messages

Commands (3)

/xchat - Main command for the script (Sub-commands: reload, help, perms)

/chat - Main chat moderation command

/xchat - Reset all xChat variables (WARNING: This returns the system to its original state. It should only be used in emergencies - requires * permission)

Permissions (17)

xchat.help - Access to /xchat help

xchat.perms - Access to /xchat perms

xchat.reload - Access to /xchat reload

xchat.chat - Access to /chat

xchat.chat.mute - Access to /chat mute

xchat.chat.unmute - Access to /chat unmute

xchat.chat.clear - Access to /chat clear

xchat.chat.filter - Access to /chat filter

xchat.chat.links - Access to /chat links

xchat.chat.cooldown - Access to /chat cooldown

xchat.chat.caps - Access to /chat caps

xchat.bypass.mute - Bypass chat mute

xchat.bypass.clear - Bypass chat clear

xchat.bypass.filter - Bypass chat filter

xchat.bypass.links - Bypass link filter

xchat.bypass.cooldown - Bypass chat cooldown

xchat.bypass.caps - Bypass caps limit
